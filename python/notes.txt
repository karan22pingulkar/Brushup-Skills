methods in python that return something
count()
Returns the number of occurrences.
nums = [1, 1, 2, 1]

result = nums.count(1)

print(result)

get()

Returns the value for a dictionary key.
d = {"name": "John"}

result = d.get("name")

print(result)
output :
John



Returns the position of the first occurrence.
| Method     | Returns                            |
| ---------- | ---------------------------------- |
| `sorted()` | New sorted list                    |
| `pop()`    | Removed element                    |
| `get()`    | Dictionary value (or default/None) |
| `count()`  | Number of occurrences              |
| `index()`  | Position of first occurrence       |

Objects whose values can be changed after creation.
